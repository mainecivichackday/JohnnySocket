JohnnySocket

Arduino Over The Interwebs

This project is an effort to explore using the browser as an interface to controlled remote hardware.

Johnny-Five

Use the Johnny-Five Node.js module to control your arduino board. The node server will run on a PC of RaspberryPi connected to the arduino via USB.

TCP

Use Node's built in TCP module to connect your local Johnny-Five instance to the webserver.

Socket.io

Use socket.io to hook button clicks on a webpage into events that your webserver can transmit to your Johnny-Five instance via TCP.
